    Python for Universal Windows Platform (UWP)

    This is a repository for CPython 3.x which can run on the Universal Windows Platform (UWP) for Windows 10 IoT Core. This is based on a GitHub fork of the https://hg.python.org/cpython/ Mercurial repository.

    The extensions in this fork are changes to CPython to compile under Windows Universal project systems.

    To support running on the IoT platform, additional tools have been developed under Tools\pyuwp\pyuwp.sln to serve as an APPX host for running Python code in a background application.
